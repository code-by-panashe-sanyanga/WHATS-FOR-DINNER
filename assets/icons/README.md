# Social Media Icons

Place your social media icon files here in SVG format:

- `instagram.svg` - Instagram icon
- `x.svg` - X (Twitter) icon  
- `facebook.svg` - Facebook icon

## Recommended Sources

- **FontAwesome Free**: https://fontawesome.com/download
- **Material Design Icons**: https://fonts.google.com/icons
- **Feather Icons**: https://feathericons.com
- **Simple Icons**: https://simpleicons.org

## Icon Naming

Keep the filenames exactly as listed above so they match the HTML references across all pages.
